# Modern Business Website

## Project Overview
A modern, responsive business website built using React.js.
The project focuses on UI/UX, responsiveness, accessibility, and performance.

## Technologies Used
- React 18
- JavaScript (ES6+)
- CSS Flexbox & Grid
- Git & GitHub

## Features
- Responsive navigation bar
- Theme toggle (dark/light mode)
- Modern hero section
- Services section
- Clean component structure

## Folder Structure
- src/ → React components and pages
- public/ → Static assets
- docs/ → Project documentation
- tests/ → Test files
- lighthouse/ → Performance reports
- .github/workflows/ → CI/CD pipeline
